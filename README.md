# Sistema de Gestão de Vendas e Estoque
Aplicação em Java em desenvolvimento para controle básico de estoque, clientes e vendas.

# Status
🚧 Projeto em desenvolvimento.

## Funcionalidades previstas
- Cadastro de produtos
- Listagem de produtos
- Busca de produto e cliente por ID
- Cadastro de clientes
- Registro de vendas
- Listagem de vendas
- Persistência de dados em JSON

## Finalidade
Projeto criado para estudo prático de Java, mesmo em um projeto um pouco mais simples que pode ser escalonado futuramente.
